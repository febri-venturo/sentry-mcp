Tampilkan pesan berikut ke user:

---

**Sentry Docs tidak tersedia di self-hosted instance.**

Fitur pencarian dokumentasi hanya tersedia untuk Sentry cloud (sentry.io). Karena instance ini menggunakan self-hosted Sentry, fitur ini tidak bisa digunakan.

Sebagai alternatif, kamu bisa akses dokumentasi langsung di:
- **Sentry Docs**: https://docs.sentry.io
- **PHP SDK**: https://docs.sentry.io/platforms/php/
- **Laravel**: https://docs.sentry.io/platforms/php/guides/laravel/
- **Golang**: https://docs.sentry.io/platforms/go/
- **JavaScript/Node**: https://docs.sentry.io/platforms/javascript/
- **Next.js**: https://docs.sentry.io/platforms/javascript/guides/nextjs/
