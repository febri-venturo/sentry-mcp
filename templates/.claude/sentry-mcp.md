# Sentry MCP Configuration

Konfigurasi ini dibaca oleh slash commands Sentry. Ganti nilai sesuai project kamu.

- **Organization Slug**: YOUR_ORG_SLUG
- **Project Slug**: YOUR_PROJECT_SLUG
- **Region URL**: https://YOUR_SENTRY_HOST
- **Seer**: disabled (self-hosted, tidak tersedia)

## Catatan
- Selalu gunakan `regionUrl` di setiap pemanggilan MCP tool
- Sentry dashboard: https://YOUR_SENTRY_HOST/organizations/YOUR_ORG_SLUG/issues/?project=YOUR_PROJECT_SLUG
